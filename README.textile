h1. rah_debug

"Packagist":https://packagist.org/packages/rah/rah_debug | "Issues":https://github.com/gocom/rah_debug/issues

Display Textpattern site's debugging information to selected logged-in users no matter what the site's production status is set to. This debugging information includes error messages and the tag trace. The plugin also adds the familiar tag trace to admin-side pages.

h2. Install

Using "Composer":https://getcomposer.org:

bc. $ composer require rah/rah_debug

h2. Basics

Rah_debug displays debugging information based on the user's group and privileges. By default, all *Publishers* will see the debugging information regardless the production status.
